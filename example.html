DeSO Address Lookup Widget Example

This page demonstrates how to embed the DeSO widget into your website. The widget appears below.

Integration Code

To add this widget to your site, include these scripts and add a div with id="deso-widget-root":

<!-- React and ReactDOM -->
<script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>

<!-- Widget loader -->
<script src="widget-loader.js"></script>

<!-- Container for widget -->
<div id="deso-widget-root"></div>
